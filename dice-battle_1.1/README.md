
# 🎲 骰子大作战 (Dice Battle)

一款基于 React + Tailwind CSS + Framer Motion 开发的策略合成消除游戏。
在高科技战术终端的包装下，挑战你的运气与策略极限！

## 🚀 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 构建生产版本
```bash
npm run build
```

## 🎮 核心玩法

- **合成进化**：滑动屏幕移动所有骰子，相同数值的骰子碰撞会合成更高级别的骰子。
- **三重裂变**：三个同级骰子同时碰撞，可直接跨级进化。
- **狂暴模式**：达成 3 连击（Combo）后开启，得分翻倍！
- **终极清理**：当[ULTIMATE]红核充满屏幕时，达成 3 连消是唯一的生存手段。

## 🛠️ 技术栈

- **框架**: React 19
- **样式**: Tailwind CSS
- **动画**: Framer Motion
- **图标**: Lucide React
- **部署**: 推荐使用 Vercel
